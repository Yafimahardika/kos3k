⚙️ Konfigurasi Firebase

Masukkan konfigurasi Firebase project Anda. Data ini disimpan di browser Anda saja.

💡 Cara mendapatkan konfigurasi:
1. Buka console.firebase.google.com
2. Buat project "koskita-app"
3. Project Settings → General → Your apps → Web app
4. Salin nilai dari firebaseConfig
5. Aktifkan Firestore Database di menu Build
